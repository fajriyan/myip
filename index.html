About

MYIP adalah sebuah project Pribadi berupa Landing Page dengan Menggunakan ✨Alphine JS dan 🍃Tailwind CSS yang digunakan untuk Melihat IP Public yang sedang digunakan serta Informasi didalamnya. Website ini menggunakan API Milik bigdatacloud.com yang memberikan data mengenai IP yang digunakan saat ini. Project ini memiliki tujuan sebagai 📚Portfolio saya sendiri yang bebas digunakan siapapun di 👩‍💻github/fajriyan. Terimakasih semoga Bermanfaat.
